# 462. Пълен с благости

_С. П. 88, Д. П. 238, Х. П. 122_

1. Пълен с благости,  
Пълен с радости  
Е денът на Рождество!  
Силите небесни,  
В хорове чудесни,  
Възвестиха тържество.  

2. Пълен с благости,  
Пълен с радости  
Е денът на Рождество!  
Грешниче, смири се,  
Господ днес роди се, -  
В този ден на тържество.  

3. Пълен с благости,  
Пълен с радости  
Е денът на Рождество!  
Днес дойде Спасител,  
Божий Примирител,  
Братя, днес е тържество!
