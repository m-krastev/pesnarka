# 211. Който на Господа

_С. П. 579, Д. П. 182_

1. Който на Господа  
Сърдечно се надей,  
Като Сионската гора  
Непоклатим ще е.  

2. Както светият град  
Е окръжен с гори,  
Той окръжава с благодат  
Бог людиете Си.  

3. Ако, като Отец,  
И да наказва Той,  
Пак милостив е; до конец  
Не пази гнева Свой.  

4. Помилвай, Боже наш,  
Своите Си слуги,  
Чиито вяра и любов  
Пред Теб са искрени.
