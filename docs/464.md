# 464. Както еленът извори

_С. П. 405_

1. Както еленът извори  
На живата вода,  
желае, Боже, Теб  
    Примрялата душа.  

2. Пред Тебе, Боже в Твоя храм  
Кога ще се явя,  
а Те славословя там  
    С безкрайна веселба?

3. Въздишам за честития  
Преминал весел ден,  
то пред лицето Ти,  
    О, Боже, бях блажен.  

4. Душо, отпаднала ли си?  
На Бога се надей:  
е спасението ти,  
    Ти Нему с радост пей.
