# 175. Когато пред престола Ти

_С. П. 391, Д. П. 135_

1. Когато пред престола Ти  
Се кланяме в молба  
Дарувай, Боже, Духа Си,  
Дух на истината.  

2. Кога за греховете си  
Търсим прощение,  
О, дай да ги усещаме  
И мразим от сърце.  

3. Когато просим милости,  
Ако ни ги дадеш,  
Да хвалим Теб, да хвалим пак  
Кога ги отречеш.  

4. Нашата воля нека е  
По волята Твоя;  
Да припознаваме в двете  
Същата доброта.
